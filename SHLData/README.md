# Explenation of feature extraction:

1. Download the training and testing data of the 2018 SHL challange from the website of the SHL challenge. The link to the 2018 data is: http://www.shl-dataset.org/activity-recognition-challenge/
2. Extract the following files of the training and testing data: Acc_x.txt, Acc_y.txt, Acc_z.txt, Gyr_x.txt, Gyr_y.txt, Gyr_z.txt, Mag_x.txt, Mag_y.txt, Mag_z.txt
3. Store them in one of the corresponding directories: '/SHLData/2018/train/imu/' or '/SHLData/2018/test/imu/'
4. Extract the file: Label.txt for training and testing data.
5. Store them in one of the corresponding directories:'/SHLData/2018/train/' or '/SHLData/2018/test/'
